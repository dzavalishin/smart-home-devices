URLs

  • /item.cgi?name=adc0 - read item value
  • /item.cgi?name=dig0&value=1 - write item value

Item names

  • dig - Digital I/O, bits
  • temp - Temperature sensors (1-wire), ordered (see Config page)
  • utemp - Temperature sensors (1-wire), unordered
  • pwm - Analogue outputs

Pins

  • PB0-3 - front panel LEDs
  • PB4-5 - relays (move to 4,6)
  • PB6 -
  • PB7 - LCD backlight PWM
  • PD0-3 - front panel buttons
  • PD4-5 -
  • PD6-7 - Di
  • PD6-7 - Encoder rotation
  • PE4 - Encoder button
  • PE3 -
  • PE2 - R/S
  • PE1 - R/W
  • PE0 - E
  • PF0-7 - LCD data bus
  • PG3 -
  • PG4 - 1Wire (2401, single bus 18B20, etc)

Modbus registers, 0 based (add 1 for modpoll)

  • 0-3 - text id, 'DZ' 'DZ' 'WA' '00'
  • 4-6 - 6 bytes of serial number (part of MAC, usually)
  • 16-19 - Di/Do: MCU ports B, D, E, F
  • 64-71 - Ai
  • 80-87 - Ao (PWM)
  • 512-518 - Config: Di count, Do count, Ai+special dev count, Ao/PWM count, counters count, temp sensors max, temp sensors actual
  • 1024... - temperature sensors, ordered by permanent ID (as configured)
  • 2048... - temperature sensors, natural order (as found)

Defaults

  • IP address - 192.168.88.147
  • MAC - 02:06:98:00:00:00
  • Serial speed - 38400
  • Debug serial - UART1 (PD2-3)
  • Error/unread value - 0x1000 for 16 bit values, 0x10000000 for 32 bit
